◆ align_loops

std::uint16_t gemmi::cif::WriteOptions::align_loops = 0

Maximum column width in loops when aligning (0=no alignment). If non-zero, all columns are padded to at most this width. This produces more compact, readable loop output.

◆ align_pairs

std::uint16_t gemmi::cif::WriteOptions::align_pairs = 0

Width reserved for tags in pairs (0=no alignment, typical value 33-34). If set, values start at column (align_pairs + 1). Example: align_pairs=33 starts values at column 35.
