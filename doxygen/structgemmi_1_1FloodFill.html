Detailed Description

template<typename T, int Land>
struct gemmi::FloodFill< T, Land >

Flood fill algorithm for finding connected regions in a 3D grid.

Template Parameters
TGrid data type (typically int8_t for binary masks)
LandValue marking the region to fill (0 or 1); fills connected regions with this value

Implements a scanline-based flood fill that respects periodic boundary conditions and uses 6-way (face) connectivity. The algorithm identifies all grid points with value Land that are connected to a seed point.
