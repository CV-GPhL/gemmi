Detailed Description

A tabular loop structure: tags (column names) and flat row-major values.

In CIF syntax, a loop is a compact representation of a table with named columns:

loop_
_category.tag1 _category.tag2 _category.tag3
value1a value2a value3a
value1b value2b value3b

Internally, the tag names are stored in tags and all values are stored sequentially in values using row-major layout: for N columns and M rows, values.size() == N*M, and element at (row r, column c) is at index r * N + c.
